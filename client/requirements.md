## Packages
date-fns | Formatting departure and arrival times into human-readable strings
framer-motion | Smooth entry animations and layout transitions for dashboard
lucide-react | Icons for the interface

## Notes
- The application automatically polls the MBTA API every 30 seconds for live data
- Uses MBTA official Red Line color (#DA291C) as the primary brand color
- Settings changes invalidate the predictions query to immediately recalculate walk times
